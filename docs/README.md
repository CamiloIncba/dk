## Documentación (DK)

Esta carpeta contiene documentación operativa y de arquitectura para el fork Dark Kitchen.

### Documentos base

- `SRS.md`: requisitos (borrador hasta aprobación; luego inmutable)
- `PLAN.md`: plan de trabajo (borrador hasta aprobación; luego inmutable)
- `ARCH.md`: arquitectura y decisiones principales
- `RUNBOOK.md`: operación (deploy, variables, incidentes)
- `PR_CHECKLIST.md`: gates de calidad
- `ADR/`: decisiones (Architecture Decision Records)
